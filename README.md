## Supported Platforms

- Apple Silicon Mac

## dependencies
- brew
- wget

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install wget
```

```sh
bash <(wget -qO- https://raw.githubusercontent.com/TakanoTaiga/ros2_m1_native/master/setup.sh)
```
